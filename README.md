# Sam Halsall Portfolio

A simple personal portfolio site built to showcase flagship side projects, technical skills, and contact links.

This project is intentionally lightweight: it acts as a clean overview and signpost to deeper project work, rather than a complex app itself.

## ✅ What this project includes

-   Hero/introduction section
-   Featured and client project highlights
-   Technical skills section
-   Experience summary
-   Contact section with personal links
-   Smooth section animations for better presentation

## 🛠️ Tech stack

-   Next.js (App Router)
-   Next MDX
-   React
-   TypeScript
-   Tailwind CSS
-   Framer Motion

## 💻 Getting started

### 1) Install dependencies

```bash
npm install
```

### 2) Run locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### 3) Production build

```bash
npm run build
npm run start
```

## 🛠️ Available scripts

-   `npm run dev` – start local dev server
-   `npm run build` – create production build
-   `npm run start` – run production server
-   `npm run lint` – run ESLint checks

## 🗂️ Project structure

-   `src/app` – Next.js app entry (`layout.tsx`, `page.tsx`, global styles)
-   `src/components/common` – core portfolio sections
-   `src/components/layout` – structural and reusable layout pieces
-   `src/components/ui` – small reusable UI elements
-   `src/components/icon` – custom SVG/icon components
-   `src/lib/constants` – navigation, links, and content constants
-   `src/lib/utilities` – utility helpers
-   `public/assets` – static assets

## 📝 Notes

-   Content is mostly driven by constants/components, so updates are straightforward.
-   This repository focuses on presentation and discoverability of other projects.

## 🚀 Deployment

Deploy on Vercel (recommended for Next.js) or any platform that supports Node.js.
